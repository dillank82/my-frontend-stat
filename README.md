# My Frontend Stat
## Описание
Веб-приложение разработанное для объективации и визуализации личного прогресса в изучении фронтенд-разработки.
## Демонстрация
https://dillank-frontend-stat
## Технологии
Frontend: React, Redux Toolkit, Vite, Jest (юнит-тестирование).
Backend: Node.js, Express, MongoDB.
Типизация: TypeScript.
Организация: pnpm Workspaces (Монорепозиторий)
## Почему я это сделал?
Проект был создан для решения проблемы мотивации и объективной оценки прогресса во время самостоятельного изучения веб-разработки, а также для демонстрации навыков работы с основным стэком технологий и практического обучения разработке бэкенда.
## Особенности реализации
- Приложение представляет собой монорепозиторий, управялемый с помощью pnpm workspaces для эффеективного разделения фронтенда и бэкенда;
- Комбинация обычных редукс слайсов с api формирует реактивный пользовательский интерфейс с динамичными данными;
- Процентный круг реализован через svg элемент с применёнными фильтрами для стилизации;
- Применение вариативных и переиспользуемых компонентов;
- Автоматическое позицонирование элементов через Flexbox.
## Как запустить
Команды:
1) Установка зависимостей: 
```pnpm i```
2) Локальный запуск:
```pnpm start```
